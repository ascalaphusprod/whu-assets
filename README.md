# WHU Assets

Visual and audio assets for the Wardbridge Heights University Board — an original fictional university worldbuilding and interactive storytelling project.

This repository contains character artwork, UI elements, campus imagery, and related media assets used in the WHU Board web project.

## Project

Wardbridge Heights University (WHU) is an original fictional setting — a private university in Wardbridge, Massachusetts, with its own characters, history, culture, and lore. The WHU Board is an interactive web experience built around this world.

## Asset Categories

- `/characters/` — Character artwork, turnarounds, and reference sheets
- `/ui/` — Logos, mascots, card templates, and interface elements
- `/world/` — Campus and town imagery
- `/audio/` — WALR 89.1 radio station audio (voiceovers, music)
